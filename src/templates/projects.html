Personal Website

I wrote my website from scratch, and the only javascript library I use is jQuery. In fact, the only other library I use is font-awesome. My website is hosted on Google App Engine, from a server I wrote in Go. The single page is actually a few files stitched together using Go's html/template package. The songs and places in the about section are stored in a database and cached server side, retrieved via API call.

Project Euler

Project Euler is a website that contains many interesting math problems that can be solved computationally. While the first few problems can be brute-forced, many require thinking and are actually quite time consuming. Being in both the engineering and math faculties, I enjoy these challenges. I have solved a bunch of them.

Minesweeper

I wrote Minesweeper in Java for a school project with a partner. This was done as a culminating assignment and combined multiple core CS concepts such as polymorphism/inheritance, GUIs (Swing), layouts (GridBagLayout), UML, and testing.

GetStuffDone

A ToDo app written for Android to keep track of a prioritized ToDo list. I implemented this using a SQLite database. This was my first exposure to mobile development, which introduced me to the activity lifecycle and responsive development.

Tetris

Tetris written in Java, with many options such as board size and game speed.

SnakeAndroid

Snake written for Android. Timeless game, bringing back fond memories. Disclaimer: this is inferior to Snake on old Nokias. Developing an Android game taught me about threading, as I needed to update the game and UI asynchronously. I also delved further into layout development with a dynamically changing UI.
